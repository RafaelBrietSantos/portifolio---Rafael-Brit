# Portfólio Profissional - Rafael Briet 🚀

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido como Atividade Prática para a disciplina de **Fundamentos da Programação Web** do bacharelado em Engenharia de Software (UNINTER).

O objetivo principal deste projeto é aplicar e demonstrar conhecimentos fundamentais de desenvolvimento frontend utilizando apenas as tecnologias base da Web, com foco em estruturação semântica, design responsivo e manipulação de comportamento via scripts.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído **sem o uso de frameworks** (como Bootstrap ou Tailwind), garantindo o domínio total das linguagens estruturais e de estilização:

* **HTML5:** Estruturação utilizando tags semânticas (`<header>`, `<main>`, `<section>`, `<article>`, etc.) para melhor acessibilidade e SEO.
* **CSS3:** Estilização com variáveis globais, paleta de cores estilo *Dark Mode/Underground*, e layouts construídos com **Flexbox** e **CSS Grid**. Totalmente responsivo utilizando *Media Queries*.
* **JavaScript (Vanilla):** Implementação de comportamento dinâmico e validação de dados no formulário de contato, interceptando o evento de envio (`preventDefault`).

## ⚙️ Funcionalidades e Estrutura

O portfólio segue o modelo *Single Page Application* (SPA) com navegação por âncoras e está dividido em quatro seções principais:

1.  **Sobre Mim:** Minha apresentação profissional, destacando a mentalidade de alto rendimento e foco analítico.
2.  **Formação:** Resumo acadêmico e cursos complementares focados em backend (Python, SQL, Git).
3.  **Portfólio:** Galeria de projetos práticos desenvolvidos.
4.  **Contato:** Formulário de comunicação funcional com validação de campos obrigatórios via JavaScript.
